Transmisor inalámbrico para deformímetros mitutoyo

Este dispositivo le permitirá transmitir las mediciones de su mitutoyo en tiempo real de una manera completamente automática sin necesidad de oprimir ningún botón del deformímetro para realizar la transmisión de las mediciones.

La adquisición de los datos se realiza mediante una señal bluetooth, y los datos se presentan tal cual aparecen en la pantalla del deformímetro, ya sea la medición en milímetros o pulgadas.

Comunicación bluetooth

Este transmisor cumple con las especificaciones del estándar bluetooth 2.0 con una frecuencia de 2.4 GHz, tiene una potencia de transmisión de 4 dBm clase 2 y el alcance de la señal puede ser de hasta 12 metros.

Este transmisor se puede enlazar a cualquier dispositivo que tenga tecnología bluetooth como computadoras, laptops, tablets y smartphones.

Vinculación

Cada transmisor tiene un código de vinculación único, por lo tanto la señal inalámbrica no puede ser corrompida y puede trabajar en conjunto con otros transmisores al mismo tiempo sin riesgo de entroncamiento con otras señales.

Una computadora o laptop puede vincular hasta 250 transmisores y soporta comunicación full-duplex con cada uno de ellos al mismo tiempo.

Programa de cómputo

Para realizar la adquisición de las mediciones desde el deformímetro es necesario ejecutar un software especializado mediante el cual usted podrá vincularse con el transmisor y solicitar los datos ya sea de manera manual o automáticamente durante un lapso de tiempo especificado por el usuario.

Soluciones IDEM pone a su alcance un programa para vincular hasta cuatro transmisores y obtener 100 lecturas de cada uno de ellos, si usted requiere un sofware personalizado contáctenos a jl_umana@solucionesidem.com
